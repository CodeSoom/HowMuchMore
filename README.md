# How Much More

## 소개
얼마나 더 많은 돈을 더 벌어야 원하는 삶을 살 수 있는지 알려드립니다. 

## 웹사이트 링크
https://howmuchmore.xyz/

## 설치하기

```bash
npm install
```

## 실행하기

```bash
npm start
```

## 린트 실행하기

```bash
npm run lint
```

## 테스트 실행하기

### 전체 테스트 실행하기

```bash
npm test
```

### 유닛 테스트 실행하기

```bash
npm run test:unit

# 파일이 저장됐을 때 자동으로 테스트 실행하기
npm run watch

# 커버리지 출력하기
npm run coverage
```
